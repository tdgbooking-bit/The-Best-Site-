# Deployment Instructions for Subliminal World Clothing Brand Website

## 1. Prepare Your Files
- Ensure all HTML, CSS, JS, and image files are in your project folder.
- Double-check all links and forms work locally.

## 2. Choose a Hosting Platform
- **Vercel** (recommended for static sites)
- **Netlify** (easy drag-and-drop or Git integration)
- **GitHub Pages** (for public projects)
- **Shared hosting** (upload via FTP)

## 3. Deploy with Vercel (Recommended)
1. Go to https://vercel.com and sign up/log in.
2. Click 'New Project' and import your GitHub repo, or drag-and-drop your folder.
3. Set the root directory to your project folder.
4. Click 'Deploy'.
5. Your site will be live at a vercel.app URL.

## 4. Deploy with Netlify
1. Go to https://netlify.com and sign up/log in.
2. Click 'Add new site' > 'Deploy manually'.
3. Drag your project folder into the drop zone.
4. Netlify will build and give you a live URL.

## 5. Deploy with GitHub Pages
1. Push your project to a GitHub repository.
2. Go to repo Settings > Pages.
3. Set source to main branch and root folder.
4. Save and visit the provided URL.

## 6. Custom Domain
- Both Vercel and Netlify allow you to add a custom domain in their dashboard.

## 7. Final Checklist
- [ ] Favicon displays in browser tab
- [ ] All navigation and forms work
- [ ] Mobile layout looks good
- [ ] Meta tags show correct info when sharing
- [ ] QR code and points system work as expected
